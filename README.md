# vintly

빈티지 커뮤니티 프론트엔드 ADMIN 프로젝트입니다.

### 개발환경

- Vue3
  - node : 22.14.0
  - yarn : 1.22.22
  - vuetify : 3.0.0

### 프로젝트 setup

```bash
# pacakge update
yarn install

# project start
yarn serve

# build
yarn build

# lint (해야하나?)
yarn lint
```
